# 命令介绍

### 列出所有的命令
```
gitbook help
```

### 帮助信息
```
gitbook --help
```

### 初始化
```
gitbook init
```

### 下载插件
```
gitbook install
```

### 生成静态网页
```
gitbook build
```

### 生成静态网页并运行服务器
```
gitbook serve
```

### 生成时指定gitbook的版本, 本地没有会先下载
```
gitbook build --gitbook=2.0.1
```

### 列出本地所有的gitbook版本
```
gitbook ls
```

### 列出远程可用的gitbook版本
```
gitbook ls-remote
```

### 安装对应的gitbook版本
```
gitbook fetch 标签/版本号
```

### 更新到gitbook的最新版本
```
gitbook update
```

### 卸载对应的gitbook版本
```
gitbook uninstall 2.0.1
```

### 指定log的级别
```
gitbook build --log=debug
```

### 输出错误信息
```
gitbook builid --debug
```
